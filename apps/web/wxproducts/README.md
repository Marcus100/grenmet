# wxproducts (`@grenmet/web-wxproducts`)

Structured meteorological products platform for GMS. Port **3005**. Own PostgreSQL database (Drizzle ORM). No `pnpm start` required.

Stores GMS forecast and observational products — morning/midday/evening forecasts, METAR/SPECI, TAF, SYNOP, CAP, IBF, BUFR, marine, tropical outlook, hourly observations. Supports Playwright PDF export.

See [CLAUDE.md](./CLAUDE.md) for the full schema structure, DB workflow, adapters, and PDF export.
See [docs/web/development.md](../../../docs/web/development.md) for startup commands.
