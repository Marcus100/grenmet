# Warning issued actions: CFR

Crash Fire Rescue should secure response assets and maintain emergency support readiness. {{ className: 'lead' }}

## Action checklist

- Secure all rescue vehicles in designated areas.
- Check all radios and flashlights.
- Liaise with Building Maintenance Foreman to secure Fire Hall areas requiring reinforcement.
- Ensure first-aid kits are properly stocked.
