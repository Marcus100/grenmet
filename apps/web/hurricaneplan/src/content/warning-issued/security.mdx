# Warning issued actions: Security

Security and Emergency Services should transition to standby posture and sustain on-duty operations through event passage. {{ className: 'lead' }}

## Action checklist

- Inform all off-duty security officers to remain on standby.
- Ensure flashlights and gas lanterns are equipped and in working order.
- Check security vehicle fuel, oil, and operating condition.
- Check all security radios.
- Instruct all security officers on duty to remain on duty until relieved.
- Make available emergency foodstuff items for duty staff.
