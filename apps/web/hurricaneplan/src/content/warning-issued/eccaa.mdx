# Warning issued actions: ECCAA

ECCAA technical specialists should execute the following equipment-protection steps during warning phase. {{ className: 'lead' }}

## Action checklist

- Secure all portable radios.
- Cover all electronic equipment including VOR/DME, NDB, receivers, and transmitters.
- Take down all VHF antennas and secure them.
