# Pre-season actions: Security

The Security Manager or designate prepares lighting, communications, and emergency support supplies before season start. {{ className: 'lead' }}

## Action checklist

- Ensure all spotlights and flashlights are serviceable.
- Ensure all security radios are serviceable.
- Prepare and maintain an emergency foodstuff list.
