# salesbus (`@grenmet/web-salesbus`)

Sales and inventory management app for GMS. Port **3007**. No database. No `pnpm start` required.

Routes: `/sales`, `/inventory`, `/settlements`, `/offline`. Currently uses mock data; `@grenmet/api-client` integration is planned.

See [CLAUDE.md](./CLAUDE.md) for cart state, UI conventions, and planned API integration.
See [docs/web/development.md](../../../docs/web/development.md) for startup commands.
