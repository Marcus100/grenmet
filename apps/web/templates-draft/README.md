# Templates Draft (`@grenmet/web-templates-draft`)

Prototype Next.js app for weather document and template drafts.

## Development

From repo root:

```bash
pnpm install
pnpm dev:web:templates-draft
```

The app runs on `http://localhost:3006`.

## Run from app directory

```bash
cd apps/web/templates-draft
pnpm dev
```

## Quality Commands

```bash
pnpm check
pnpm check:ci
pnpm type-check
```
