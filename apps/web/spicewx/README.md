# SpiceWx (`@grenmet/web-spicewx`)

Next.js weather app in the Grenmet monorepo.

## Development

From repo root:

```bash
pnpm install
pnpm dev:web:spicewx
```

The app runs on `http://localhost:3004`.

## Run from app directory

```bash
cd apps/web/spicewx
pnpm dev
```

## Quality Commands

```bash
pnpm check
pnpm check:ci
pnpm type-check
```
