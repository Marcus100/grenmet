# Hono API (`@grenmet/api-hono`)

Simple Hono service in the Grenmet monorepo.

## Run (recommended)

From repo root:

```bash
pnpm install
pnpm dev:honoapi
```

The service listens on `http://localhost:3000`.

## Run from app directory

```bash
cd apps/api/honoapi
pnpm dev
```

## Other scripts

```bash
pnpm build
pnpm start
pnpm type-check
pnpm check:ci
```
