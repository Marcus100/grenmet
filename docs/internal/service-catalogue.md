# GMS Service and Product Catalogue

This document defines the full catalogue of potential GMS services and their associated products. It is the detailed operational reference that accompanies the strategic framing in [GMS Digital Service Architecture](./architecture.md).

> **Status:** Planning and governance reference. These are potential services, not a committed build order. Each service and product must be phased, resourced, and approved by GMS leadership and relevant stakeholders before it becomes an operational commitment. The [programme roadmap](./internal/roadmap.md) tracks what is actively in scope.

## A. Public Weather Service

The general national forecast service for the public, media, schools, tourism sector, road users, and general daily planning.

**Main purpose:** Help the public understand expected weather, hazards, and practical actions.

| Product | Description |
|---|---|
| Daily Public Forecast | General national forecast for Grenada, Carriacou, and Petite Martinique |
| 3-Day Forecast | Short-range outlook for planning |
| 7-Day Outlook | Lower-detail extended public forecast |
| Morning Weather Briefing | Early-day summary for public and media |
| Evening Weather Summary | Summary of current conditions and overnight outlook |
| Weekend Weather Outlook | Public-facing forecast for weekend activities |
| Weather Graphic Card | Social media / website visual forecast |
| Key Messages Graphic | 3–5 main public messages during significant weather |
| Weather Explainer | Simple educational posts: Saharan dust, troughs, tropical waves, etc. |

## B. Warning and Impact-Based Forecasting Service

One of GMS's most important services. Warnings should use impact-based language: not only what the weather will be, but what it may do.

**Main purpose:** Warn people and agencies about hazardous weather with clear impacts and actions.

| Product | Description |
|---|---|
| CAP Alert | Structured official alert for multi-channel dissemination |
| Heavy Rainfall Advisory | Lower-level rainfall risk product |
| Heavy Rainfall Watch | Conditions possible; users should monitor |
| Heavy Rainfall Warning | Hazard expected or occurring |
| Flash Flood Advisory / Watch / Warning | Flood-specific public warning product |
| High Wind Advisory / Warning | For strong winds outside tropical cyclone context |
| Severe Weather Bulletin | General warning bulletin for multi-hazard events |
| Impact Matrix | Likelihood × impact table for decision-makers |
| Parish / Zone Impact Map | Localised impact forecast by area |
| Key Messages Bulletin | Short, action-focused hazard summary |
| Preparedness Instructions | Standardised "what to do now" content |
| Post-Event Warning Review | Verification and lessons learned after major events |

### Recommended risk structure

| Dimension | Levels |
|---|---|
| **Likelihood** | Very Low · Low · Medium · High · Very High |
| **Impact** | Minor · Moderate · Severe · Extreme |
| **Response** | Be Aware · Be Prepared · Take Action |

This aligns with the IBF framework GMS is already moving toward. See [architecture.md — Draft Warning Model](./architecture.md#draft-warning-model) for the full warning content contract and CAP mapping.

## C. Tropical Cyclone Service

GMS should not try to replace NHC. The value of GMS is national interpretation of regional guidance for Grenada-specific conditions.

```
NHC official guidance
        ↓
GMS local interpretation
        ↓
Grenada-specific impacts
        ↓
CAP alert / bulletin / graphics / briefings
```

**Main purpose:** Translate regional tropical cyclone guidance into Grenada-specific risk, impacts, and actions.

| Product | Description |
|---|---|
| Grenada Tropical Weather Outlook | Daily hurricane-season outlook focused on threats to Grenada |
| Special Tropical Weather Outlook | Issued when a disturbance requires attention outside routine schedule |
| Tropical Cyclone Public Advisory | Plain-language update for public |
| Tropical Cyclone Technical Discussion | Internal / professional reasoning product |
| Tropical Storm / Hurricane Watch Bulletin | Local watch product |
| Tropical Storm / Hurricane Warning Bulletin | Local warning product |
| Arrival Time of Hazardous Winds Graphic | When dangerous winds may begin locally |
| Local Wind Impact Graphic | Expected wind impacts by parish / zone |
| Rainfall and Flood Impact Bulletin | Rainfall / flood risk from tropical systems |
| Storm Surge / Coastal Flooding Bulletin | Coastal hazard risk where relevant |
| Tropical Cyclone Key Messages Graphic | Public-facing summary |
| Agency Decision Briefing | For NDEMA, GAA, Cabinet, ministries, utilities |
| Tropical Cyclone Situation Report | Operational update during active events |
| Post-Tropical Cyclone Report | Event summary, observations, impacts, forecast performance |
| Seasonal Hurricane Summary | End-of-season national summary |

## D. Aviation Meteorological Service

A core service because GMS directly supports Maurice Bishop International Airport and aviation operations.

**Main purpose:** Support flight safety, airport operations, air traffic services, airlines, pilots, and regulators.

| Product | Description |
|---|---|
| METAR | Routine aviation weather observation |
| SPECI | Special aviation weather observation |
| TAF | Aerodrome forecast |
| Trend Forecast | Short-term landing forecast where applicable |
| Aerodrome Warning | Hazard warning for airport operations |
| Wind Shear Alert | Low-level wind shear risk |
| SIGMET Support / Coordination | For significant en-route weather, depending on regional responsibility |
| Aviation Weather Briefing | Briefing for pilots, ATC, airport operations |
| Convective Outlook for Aviation | Thunderstorm / lightning risk near airport |
| Low Visibility Advisory | Fog, haze, dust, heavy rain visibility concerns |
| Crosswind / Runway Impact Briefing | Decision support for airport operations |
| Aviation Climatology Report | Wind, rainfall, visibility, thunderstorm statistics for planning |
| Aviation Incident Weather Report | Weather reconstruction for operational / safety investigations |

### Aviation Weather Dashboard (digital opportunity)

A consolidated aviation dashboard could provide:

- Live METAR / SPECI
- Current TAF
- Radar / satellite
- Lightning
- Wind / visibility trends
- Active alerts
- Runway-relevant wind components
- Aviation briefing archive

## E. Marine Weather Service

Grenada has significant marine service needs across fishing, inter-island travel, yachting, tourism, small craft, ports, and coastal communities.

**Main purpose:** Support safe marine operations and coastal decision-making.

| Product | Description |
|---|---|
| Marine Forecast | Wind, waves, weather, visibility, sea state |
| Small Craft Advisory | Hazardous conditions for small vessels |
| Small Craft Warning | More severe marine hazard |
| High Surf Advisory | Dangerous breaking waves near shore |
| Swell Forecast | Swell height, period, direction |
| Coastal Waters Forecast | Nearshore / localised marine forecast |
| Offshore Waters Forecast | Broader marine zones |
| Fisherfolk Weather Briefing | Plain-language marine decision support |
| Yachting / Marina Forecast | Tourism / marina-focused product |
| Marine Hazard Graphic | Map / card for sea state and warnings |
| Coastal Flooding Bulletin | Coastal inundation risk |
| Marine Incident Weather Report | Weather reconstruction for marine events |

### Suggested marine zones

The following zones are indicative and should be defined operationally with marine and fisheries stakeholders:

- Grenada west coast
- Grenada east coast
- Carriacou and Petite Martinique
- Coastal waters (within x nautical miles)
- Open Atlantic waters
- Caribbean Sea side
- Offshore waters (beyond x nautical miles)

## F. Climate Service

Different from daily forecasting. Deals with historical data, statistics, trends, normals, extremes, and planning.

**Main purpose:** Support agriculture, water management, infrastructure, disaster risk reduction, research, insurance, tourism, and national planning.

| Product | Description |
|---|---|
| Monthly Climate Summary | Rainfall, temperature, extremes, anomalies |
| Annual Climate Summary | Yearly national climate report |
| Rainfall Bulletin | Monthly / seasonal rainfall performance |
| Drought Monitor | Drought status and risk |
| Seasonal Rainfall Outlook | Probabilistic rainfall forecast |
| Dry Season Outlook | Dry season risk product |
| Wet Season Outlook | Wet season risk product |
| Heat Summary / Outlook | Temperature and heat stress information |
| Climate Normals | 30-year baseline statistics |
| Extreme Weather Statistics | Return periods, maximum rainfall, wind extremes |
| Climate Data Request Report | Custom report for users |
| Sector Climate Briefing | Agriculture, water, health, tourism, energy |
| Climate Change Summary | National climate trends and projections |
| Station Metadata Catalogue | Information on observing stations and data quality |

## G. Agriculture Weather Service

A high-value national service, especially if GMS links meteorology with food security.

**Main purpose:** Help farmers make decisions on planting, irrigation, spraying, harvesting, pest/disease risk, and drought preparedness.

| Product | Description |
|---|---|
| Agrometeorological Bulletin | Weather and crop-relevant forecast |
| Rainfall Outlook for Farmers | Simple rainfall guidance |
| Dry Spell Advisory | Consecutive dry-day risk |
| Irrigation Advisory | Based on rainfall, evapotranspiration, and soil moisture if available |
| Spraying Weather Forecast | Wind / rain guidance for pesticide / fertilizer application |
| Heat Stress Advisory for Livestock | Temperature / humidity-based risk |
| Pest / Disease Weather Risk Outlook | Weather conditions favourable for pests and disease |
| Soil Moisture / Drought Bulletin | If data / modelling capacity exists |
| Farmer SMS / WhatsApp Forecast | Short practical forecast for farmers |

## H. Hydrometeorological and Flood Support Service

Overlaps with warnings but deserves its own service if GMS supports flood risk, rainfall monitoring, river basins, and drainage impacts.

**Main purpose:** Support flood preparedness, drainage management, emergency response, and infrastructure planning.

| Product | Description |
|---|---|
| Rainfall Monitoring Dashboard | Real-time or near-real-time rainfall observations |
| Heavy Rainfall Outlook | Risk over next 24–72 hours |
| Flash Flood Guidance Bulletin | Flood potential based on rainfall and antecedent conditions |
| Urban Flood Risk Bulletin | Road / drainage impact focus |
| Landslide Risk Outlook | For vulnerable slopes if data exists |
| Rainfall Accumulation Map | 1h, 3h, 6h, 12h, 24h accumulations |
| Rainfall Return Period Report | How unusual an event was statistically |
| Post-Flood Weather Report | Rainfall verification and impacts |

## I. Disaster Risk and Emergency Management Decision-Support Service

The service for NDEMA, Cabinet, ministries, police, fire, health, utilities, ports, airports, and local government.

**Main purpose:** Provide decision-ready meteorological intelligence before and during hazardous events.

| Product | Description |
|---|---|
| Agency Weather Briefing | Formal briefing for emergency stakeholders |
| Decision Timeline | Expected hazard onset, peak, and improvement |
| Confidence Assessment | Forecast confidence and uncertainty |
| Scenario Briefing | Best case / most likely / worst case |
| Multi-Hazard Impact Briefing | Rain, wind, seas, flooding, lightning, heat, dust |
| EOC Weather Desk Update | Emergency Operations Centre-specific operational product |
| Situation Report Input | Weather section for national sitreps |
| Post-Event Technical Review | Verification, gaps, and recommendations |
| Preparedness Trigger Matrix | Links weather thresholds to agency actions |
